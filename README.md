# 📊 Wholesale Banking Sales Pipeline & Forecasting Dashboard

An executive command-center style Tableau dashboard for pipeline health, revenue forecasting, stale opportunity tracking, relationship-manager follow-up and weekly leadership reviews.

## 📌 Purpose of the Sales Pipeline Dashboard

Wholesale banking leaders need a weekly source of truth to monitor sales pipeline health, forecast expected revenue, identify stale opportunities, and prioritize relationship-manager follow-up before quarter-end.

This Tableau dashboard supports weekly business reviews by combining opportunity, client, product, industry, relationship-manager, and target data into governed KPIs and executive-ready views.

## 👥 Business Users

- Head of Wholesale Banking
- Sales Leadership and Enablement Team
- Finance Business Partners
- Relationship Managers
- Product Leaders
- BI / Analytics Team

## 🎯 Decisions Supported

- How healthy is the current sales pipeline?
- Which industries are showing the strongest pipeline growth?
- Which product lines need leadership attention?
- Which relationship managers have the highest stale pipeline exposure?
- Which late-stage deals are at risk before quarter-end?
- Are we likely to hit quarterly targets?
- Which clients or segments should be prioritized?

> Note: This project uses synthetic data created for portfolio purposes. No real client, employer, or confidential banking data is used.
